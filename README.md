## 🎯 What is MyPlaceholders?

**Your custom placeholders plugin for Minecraft.**

Tired of relying only on what PlaceholderAPI offers? With MyPlaceholders you can create your own placeholders with exactly what you need: vibrant colors, rotating messages, animations, and more.

Imagine a scoreboard where every player sees their rank with custom colors, or a tablist with gradient titles. That's what this plugin does.

---

## ✨ Features

| Feature | What it does |
|---------|--------------|
| 🎨 **Colors & Gradients** | Text with hexagonal colors that look amazing |
| 🔀 **Random Messages** | Shows a different message every time it updates |
| ⏱️ **Animations** | Text that changes over time (effects, loading bars) |
| 📊 **Progress Bars** | Visually displays XP, level, or any progress |
| 🎭 **Conditions** | Shows different text based on player requirements |
| 🔢 **Math** | Compares values and reacts based on results |

## 🚀 Installation

### Requirements:
- A server running **Paper** or **Spigot** (1.16 to 1.21)
- **PlaceholderAPI** installed

### Steps:

1. **Download** the `MyPlaceholders-X.X.X.jar` file
2. **Upload** it to your server's `/plugins/` folder
3. **Restart** your server
4. **Done!** The plugin will create all necessary folders automatically

```
plugins/

```
